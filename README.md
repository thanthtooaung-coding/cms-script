## Part 1: Content Management System Overview (5 minutes)

### 1.1 Landing Page and Services (1 minute)

## Screen Direction Notes:

### Key Screen Actions to Capture:

1. **CMS Admin:**

   * Dashboard statistics
   * Page request approval workflow
   * Page list with clickable URLs
   * Owner management

2. **LMS:**

   * Owner dashboard with statistics
   * Course creation and management
   * Instructor detail pages with statistics
   * Enrollment views (owner vs instructor)
   * Category management

3. **BMS:**

   * Movie listings and showtimes
   * Booking process
   * Admin dashboard

4. **ECS:**

   * Product catalog
   * Product management in dashboard
   * Shopping cart and checkout
   * Order management
   * Dashboard analytics

### Video Production Tips:

1. **Pacing**: Allow 2-3 seconds for viewers to read text on screen before speaking
2. **Zoom**: Use zoom/pan effects to highlight important UI elements
3. **Transitions**: Smooth transitions between sections
4. **Annotations**: Consider adding text annotations for key features
5. **Cursor**: Keep cursor movements smooth and purposeful
6. **Audio**: Clear narration with minimal background noise
